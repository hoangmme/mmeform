Compact WordPress form builder with external embeds, URL attribution, MME Chatbot, Google Sheets webhooks, and Twenty CRM sync.

== Installation ==

1. Upload the plugin folder to /wp-content/plugins/mme-form.
2. Activate MME Form.
3. Edit the automatically created form under MME Forms.
4. Use the generated shortcode or external embed script.
